# Currency Converter

A Java-based Currency Converter application that allows users to convert amounts between different currencies using predefined exchange rates.

## Features

* Convert currency values quickly and accurately
* Supports multiple currencies
* User-friendly console interface
* Input validation for better user experience
* Easy to extend with additional currencies

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Collections Framework

## Project Structure

```text
CurrencyConverter/

├── README.md

```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/CurrencyConverter.git
```

2. Navigate to the project directory

```bash
cd CurrencyConverter
```

3. Compile the project

```bash
javac *.java
```

4. Run the application

```bash
java Main
```

## Example

```text
Enter Amount: 100
From Currency: USD
To Currency: INR

Converted Amount: 8330.00 INR
```
